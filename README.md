Repository for individual homeworks. subject:AI_POWERED_APPS
WEEK 1
In this project, we created a Python script to interact with the Gemini language model API. The script measures input tokens, output tokens, total tokens, API latency, and estimates the cost for each request. All usage data is automatically logged in a Markdown file (cost-analysis.md) with headers and unique IDs for each API call. The logging system is robust, handling new files, empty files, and appending new rows safely with UTF-8 encoding. This setup allows easy tracking and analysis of API usage and costs over multiple runs.
